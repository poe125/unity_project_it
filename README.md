Monster Summoner

【概要】

Monster Summoner は、AR（拡張現実）を用いて遊ぶターン制カードバトルゲームです。

物理カードをスマートフォン（Android）で読み取ることで、キャラクターや攻撃が 3D モデルとして表示され、HP などのステータスを AR 上で確認できます。

カードは以下の 3 種類で構成されています：

•	キャラクターカード

•	攻撃カード

•	バフカード（攻撃力UP / HP回復）

これらを組み合わせて戦略的にバトルを進めるゲームです。

【ゲームのルール】

1\.	バトル開始

o	プレイヤーは物理カードから 2 体のキャラクターを選び、場に出します。

	Player1 のキャラ

	Player2 のキャラ

2\.	ターン制バトル

o	プレイヤーは交互にターンを行います。

o	1ターンにつき1回の行動が可能です。

3\.	選べる行動

o	攻撃

o	攻撃力バフ

o	HP回復

4\.	攻撃計算式

ダメージ ＝（キャラの基礎攻撃力 × 攻撃倍率） × 2（有利属性の場合） − 相手キャラの防御力

5\.	勝敗

o	どちらかのキャラクターの HP が 0 になった時点で負けとなります。

【カードデータベース（JSON）仕様】

カードはすべて カードID（ImageID = DataID） で管理します。

ID をキーとしてデータを読み込み、カード情報を取得します。

キャラクターカード

•	キャラ名

•	HP

•	基礎攻撃力

•	防御力

•	属性

攻撃カード

•	攻撃名

•	攻撃倍率

バフカード

•	バフ名

•	バフ効果

o	攻撃力UP：追加攻撃力（基礎攻撃力に加算）

o	回復：HPの回復量
